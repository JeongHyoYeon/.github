# AfterTrip
> 여행 후 그룹별 사진 공유의 불편함을 해결하기 위해 얼굴 인식, 객체 인식 AI를 이용해 폴더를 분류하는 사진 공유용 모바일 웹서비스

## 기능
### 시연영상
[시연영상 링크](https://youtu.be/cbFtGsFWJLc)

### 그룹 만들기 & 초대
- 아이디로 친구 초대 가능
- 그룹 페이지는 그룹 구성원들이 공통으로 보게 되는 공유 페이지
